# Color Schemes (3 Options)

Use one of these as the base design token system. All options include light/dark guidance and semantic tokens.

Brand direction: Warm, playful, expressive, social-first.

| Token | Hex | Usage |
|---|---:|---|
| `color.primary.500` | `#FF5A5F` | Primary CTA, active states |
| `color.primary.600` | `#E94A50` | Pressed CTA |
| `color.secondary.500` | `#FFB703` | Highlights, badges |
| `color.accent.500` | `#6C63FF` | Timers, special interactions |
| `color.success.500` | `#16A34A` | Success states |
| `color.warning.500` | `#F59E0B` | Warnings |
| `color.error.500` | `#DC2626` | Errors/reporting |

## Selection Guidance

- Choose `A` if you want playful and broad appeal.
- Choose `B` if you want safer/trust-first brand positioning.
- Choose `C` if voice/nightlife and premium feel are core to the brand.
